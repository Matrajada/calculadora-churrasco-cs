# 🍖 Calculadora de Churrasco

Este projeto é um programa em C# que calcula automaticamente a quantidade de alimentos e bebidas necessários para um churrasco, com base no número de adultos e crianças.

## 📌 Descrição

O sistema recebe como entrada:

- Número de adultos que consomem bebidas alcoólicas
- Número de adultos que não consomem bebidas alcoólicas
- Número de crianças

A partir dessas informações, o programa calcula a quantidade ideal de:

- Carne
- Acompanhamentos
- Cerveja
- Refrigerante
- Água

Tudo com base em uma estimativa padrão de consumo por pessoa.

## 🚀 Funcionalidades

- 📥 Entrada de dados via console
- 🧮 Cálculo automático das quantidades
- 🔢 Conversão de unidades (gramas → kg, ml → litros)
- 🎯 Arredondamento dos valores para melhor visualização
- 📊 Separação de consumo entre adultos e crianças

## 🛠️ Tecnologias Utilizadas

- C#
- .NET (Console Application)
